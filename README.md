# ML-fundamentals-2026

Repository for the **Machine Learning Foundations** course (2026).

## Assignment 1 — Data Preparation

Binary classification task using the [UCI Bank Marketing dataset](https://archive.ics.uci.edu/ml/datasets/Bank+Marketing). The goal is to predict whether a client subscribes to a term deposit following a telemarketing campaign.

### What the notebook covers

- Exploratory data analysis and target identification
- Handling explicit and implicit missing values
- Stratified train / validation / test splitting
- Categorical encoding (ordinal and one-hot)
- Feature scaling with StandardScaler
- Feature selection (variance and correlation filters)
- Class imbalance handling via random oversampling
- Logistic Regression baseline with evaluation metrics

### How to run

Open the notebook in Google Colab. The dataset is downloaded automatically from the UCI repository in the first code cell — no manual upload needed.

### Requirements

Python 3.x, pandas, numpy, matplotlib, seaborn, scikit-learn
